# Pricetunity

A website where you can find your favorite items for the lowest prices!

Check it out here: [pricetunity.com](https://pricetunity.com/)

# Dev Instructions

To run Backend:

From the project root folder, run cd Backend

1. pip install

2. venv/Scripts/activate

3. python app.py

To run Frontend:

1. npm install

2. cd Frontend

3. npm run dev
